# Marketing-ROI-Sales-Performance-Intelligence-Dashboard
An enterprise-scale Marketing ROI &amp; Sales Performance Intelligence Dashboard analyzing 100,000+ rows of marketing data using Python, SQL (BigQuery), and Power BI.
## 🎯 Business Objective
As a marketing professional with 6+ years of experience transitioning into Data Analytics, I built this end-to-end project to solve the core business challenge: **Measuring the true impact of marketing spend on revenue.** This dashboard identifies budget waste and highlights high-performing regions for better resource allocation.

## 🚀 Key Highlights
- **Data Volume:** Handled an enterprise-scale dataset of **100,000+ rows**.
- **Metrics Focus:** Developed custom KPIs for **ROAS (Return on Ad Spend)** and Revenue Contribution.
- **Domain-Led Insights:** Focused on actionable insights like identifying under-performing segments for optimization.

## 🛠️ Tech Stack & Workflow
1. **Python:** Engineered a realistic marketing dataset simulating seasonality, trends, and adstock effects (100k+ rows).
2. **SQL (Google BigQuery):** Conducted deep data cleaning, validation checks, and complex aggregations.
3. **Power BI:** Built an interactive storytelling dashboard with dynamic slicers and performance visuals.

## 📂 Project Structure
- `data_generation.py`: The Python script used to create the 100k rows CSV.
- `cleaning_queries.sql`: SQL scripts used in BigQuery for data integrity and ROI calculations.
- `Marketing_Dashboard.pbix`: The final Power BI dashboard file.
- `Dashboard_Screenshot.png`: A visual preview of the project.

## 📊 Insights Delivered
- **Top Performer:** Identified Region 3 and Product G as the highest revenue contributors.
- **Optimization Opportunity:** Pinpointed segments with ROAS < 1.5, suggesting a reallocation of the marketing budget to high-efficiency products.
